# React Hook Form + Zod

Skill для работы с формами и валидацией.

## Темы

- React Hook Form: useForm, Controller, useFieldArray
- Zod: schema definition, refinements, transforms
- Интеграция через @hookform/resolvers/zod
- Типобезопасные формы: infer типов из Zod-схемы
- Вложенные формы и динамические поля
- Server-side validation и синхронизация ошибок
- Производительность: uncontrolled inputs, minimal re-renders
- Паттерны: multi-step forms, conditional fields
