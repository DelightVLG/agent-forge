# Tailwind CSS

Skill для работы с Tailwind CSS.

## Темы

- Utility-first подход и композиция классов
- Кастомизация темы: colors, spacing, typography, breakpoints
- Responsive design: breakpoints, container queries
- Dark mode: class strategy vs media strategy
- Анимации: transition, animation, @keyframes
- Плагины: @tailwindcss/forms, @tailwindcss/typography
- Извлечение компонентов: @apply vs component abstractions
- Оптимизация: purge неиспользуемых стилей
