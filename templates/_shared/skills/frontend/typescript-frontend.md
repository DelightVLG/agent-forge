# TypeScript (Frontend)

Skill для типизации фронтенд-кода.

## Темы

- Типизация компонентов: FC, PropsWithChildren, generic components
- Типизация пропсов, событий, ref
- Discriminated unions для состояний UI (loading | error | success)
- Типизация контекстов и хуков
- Utility types: Pick, Omit, Partial, Required, Record
- Type narrowing и type guards
- Типизация API-ответов и form data
- Module augmentation для расширения библиотечных типов
