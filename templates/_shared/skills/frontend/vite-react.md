# Vite + React

Skill для работы со сборкой Vite + React.

## Темы

- Конфигурация `vite.config.ts`: plugins, resolve, build
- HMR и Fast Refresh
- Environment variables: import.meta.env
- Плагины: @vitejs/plugin-react, vite-plugin-svgr
- Proxy для API в dev-режиме
- Оптимизация сборки: chunking, tree-shaking, code splitting
- Path aliases
- Тестирование с Vitest
