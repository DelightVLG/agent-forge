# React

Skill для работы с React-приложениями.

## Темы

- Хуки: useState, useEffect, useContext, useReducer, useMemo, useCallback
- Custom hooks: извлечение логики, композиция
- Паттерны композиции: compound components, render props, HOC
- React.memo и оптимизация ре-рендеров
- Suspense, lazy loading, error boundaries
- Server Components (RSC) и клиент/сервер разделение
- Refs: useRef, forwardRef, useImperativeHandle
- Тестирование: React Testing Library, user-event
