# React Query / TanStack Query

Skill для data fetching и серверного состояния.

## Темы

- useQuery: ключи, staleTime, gcTime, refetch стратегии
- useMutation: optimistic updates, invalidation
- useInfiniteQuery: пагинация и infinite scroll
- Prefetching и SSR hydration
- Query keys: структурирование и автоматическая инвалидация
- Параллельные и зависимые запросы
- Error и loading states
- Devtools для отладки
