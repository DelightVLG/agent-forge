# App Store / Google Play

Skill для подготовки к публикации приложений.

## Темы

- iOS: App Store Connect, certificates, provisioning profiles
- Android: Google Play Console, signing keys, upload key
- Скриншоты и metadata для сторов
- Version bumping стратегии
- EAS Submit для автоматической публикации
- Review guidelines: частые причины отклонения
- TestFlight / Internal Testing tracks
- Staged rollouts и phased releases
