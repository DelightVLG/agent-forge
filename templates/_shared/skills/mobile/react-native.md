# React Native

Skill для работы с React Native.

## Темы

- Компоненты: View, Text, ScrollView, FlatList, SectionList
- Стилизация: StyleSheet, Flexbox, платформо-специфичные стили
- Платформо-специфичный код: Platform.select, .ios.tsx / .android.tsx
- Навигация и deep linking
- Нативные модули и TurboModules
- Производительность: FlatList оптимизация, InteractionManager, useMemo
- Анимации: Animated API, Reanimated
- Безопасные зоны: SafeAreaView, useSafeAreaInsets
