# React Native Testing Library

Skill для тестирования React Native компонентов.

## Темы

- render, screen, fireEvent, waitFor
- Queries: getByText, getByTestId, queryBy*, findBy*
- Тестирование навигации и экранов
- Моки нативных модулей (jest.mock)
- Тестирование custom hooks с renderHook
- Snapshot тесты: когда использовать, когда избегать
- Accessibility тесты
- Интеграционные тесты с MSW для API
