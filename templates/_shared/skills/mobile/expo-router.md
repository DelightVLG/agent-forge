# Expo Router

Skill для file-based routing в Expo.

## Темы

- File-based routing: app/ директория, соглашения именования
- Layouts: \_layout.tsx, вложенные layouts, tabs, stack, drawer
- Навигация: Link, useRouter, useLocalSearchParams
- Deep linking: автоматическая генерация схем
- Модальные экраны и параллельные routes
- Error boundaries и not-found routes
- Middleware и route guards для auth
- Типизация маршрутов
