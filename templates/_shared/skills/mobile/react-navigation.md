# React Navigation

Skill для React Navigation.

## Темы

- Stack Navigator: screens, options, transitions
- Tab Navigator: bottom tabs, material top tabs
- Drawer Navigator: кастомный drawer content
- Вложенные навигаторы и параметры между экранами
- Deep linking: конфигурация, обработка URL
- Типизация навигации: ParamList, useNavigation, useRoute
- Кастомные transitions и анимации
- State persistence и state restoration
