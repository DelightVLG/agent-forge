# Expo

Skill для работы с Expo.

## Темы

- Managed workflow vs bare workflow
- Config plugins для расширения нативного кода
- EAS Build: профили сборки, credentials
- EAS Update: OTA-обновления, каналы, rollback
- Expo SDK модули: Camera, Location, Notifications, FileSystem
- expo-modules-api для кастомных нативных модулей
- App config: app.json / app.config.ts, dynamic config
- Splash screen и app icon конфигурация
