# Drizzle

Skill для работы с Drizzle ORM.

## Темы

- Schema definition: tables, columns, relations
- Миграции: `drizzle-kit generate`, `drizzle-kit migrate`
- Type-safe queries: select, insert, update, delete
- Joins и subqueries
- Prepared statements
- Drizzle Studio для отладки
- Интеграция с PostgreSQL, MySQL, SQLite
- Batch operations и transactions
