# Redis

Skill для работы с Redis.

## Темы

- Кэширование: стратегии (cache-aside, write-through, write-behind)
- Pub/Sub: каналы, паттерны подписки
- Сессии: хранение, TTL, безопасность
- Структуры данных: strings, hashes, lists, sets, sorted sets, streams
- Rate limiting через Redis
- Distributed locks (Redlock)
- Redis Streams для очередей сообщений
- Мониторинг и оптимизация памяти
