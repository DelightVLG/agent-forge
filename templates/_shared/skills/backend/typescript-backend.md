# TypeScript (Backend)

Skill для строгой типизации серверного кода.

## Темы

- Строгий `tsconfig.json` для backend (`strict`, `noUncheckedIndexedAccess`)
- Generics и utility types для сервисного слоя
- Типизация middleware, request/response
- Branded types для ID, валют, единиц измерения
- Discriminated unions для обработки ошибок
- Type-safe конфигурация и environment variables
- Declaration merging для расширения Express/Fastify типов
