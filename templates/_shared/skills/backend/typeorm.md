# TypeORM

Skill для работы с TypeORM.

## Темы

- Entities: декораторы, columns, relations
- Repositories и custom repositories
- Миграции: генерация, запуск, откат
- Query Builder: select, join, subqueries
- Relations: eager/lazy loading, cascade options
- Subscribers и listeners
- Connection management и multiple data sources
- Тестирование с in-memory database
