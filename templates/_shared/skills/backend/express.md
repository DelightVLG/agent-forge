# Express

Skill для работы с Express-приложениями.

## Темы

- Middleware pattern и цепочки обработки
- Роутинг: Router, параметры, вложенные маршруты
- Error handling middleware
- Валидация запросов
- Статические файлы и шаблонизация
- Security middleware (helmet, cors, rate-limit)
- Тестирование через supertest
