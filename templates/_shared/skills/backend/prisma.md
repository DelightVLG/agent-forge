# Prisma

Skill для работы с Prisma ORM.

## Темы

- Schema definition: models, relations, enums, composites
- Миграции: `prisma migrate dev`, `prisma migrate deploy`
- Seeding: `prisma db seed`
- Client generation и типобезопасные запросы
- Relations: one-to-one, one-to-many, many-to-many, self-relations
- Raw queries и `$queryRaw` / `$executeRaw`
- Middleware и extensions
- Оптимизация: select/include, batch operations, connection pooling
