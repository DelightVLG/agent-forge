# GraphQL

Skill для работы с GraphQL API.

## Темы

- Schema-first vs Code-first подходы
- Resolvers: queries, mutations, subscriptions
- DataLoaders для решения N+1 проблемы
- Input validation и custom scalars
- Authentication и authorization в resolvers
- Pagination: cursor-based, offset-based
- Error handling и union types для ошибок
- Code generation: graphql-codegen для типов
