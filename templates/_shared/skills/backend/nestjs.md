# NestJS

Skill для работы с NestJS-приложениями.

## Темы

- Модульная архитектура: modules, controllers, providers
- Dependency injection и custom providers
- Guards, interceptors, pipes, exception filters
- Custom decorators
- Middleware и lifecycle hooks
- Конфигурация через `@nestjs/config`
- Валидация DTO через `class-validator` / `class-transformer`
- Тестирование: unit (Jest + Test module), e2e (supertest)
