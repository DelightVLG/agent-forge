# PostgreSQL

Skill для работы с PostgreSQL.

## Темы

- Проектирование схемы: нормализация, типы данных, constraints
- Индексы: B-tree, GIN, GiST, partial, covering
- Миграции: up/down, идемпотентность, zero-downtime
- Оптимизация запросов: EXPLAIN ANALYZE, CTE, window functions
- Партиционирование таблиц
- JSON/JSONB операции
- Транзакции и уровни изоляции
- Connection pooling (PgBouncer)
