# Docker

Skill для контейнеризации приложений.

## Темы

- Dockerfile: multi-stage builds, layer caching
- docker-compose: сервисы, volumes, networks, depends_on
- Оптимизация образов: .dockerignore, минимальные base images
- Health checks и graceful shutdown
- Environment variables и secrets
- Development containers: hot-reload через bind mounts
- Production best practices: non-root user, read-only FS
- Registry: build, tag, push workflow
