# CI/CD (GitHub Actions)

Skill для настройки CI/CD пайплайнов.

## Темы

- GitHub Actions: workflows, jobs, steps, matrix builds
- Triggers: push, pull_request, schedule, workflow_dispatch
- Кэширование: pnpm store, node_modules, Docker layers
- Secrets и environment protection rules
- Тестирование в CI: unit, integration, e2e
- Deploy workflows: staging → production, manual approval
- Артефакты: upload/download, release assets
- Reusable workflows и composite actions
