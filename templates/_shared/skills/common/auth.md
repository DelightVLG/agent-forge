# Auth (JWT / OAuth2)

Skill для реализации аутентификации и авторизации.

## Темы

- JWT: access tokens, refresh tokens, rotation
- OAuth2: authorization code flow, PKCE
- Session-based auth: cookies, secure flags, SameSite
- Роли и permissions: RBAC, ABAC
- Password hashing: bcrypt, argon2
- Multi-factor authentication (MFA/2FA)
- Social login: Google, GitHub, Apple
- Security: CSRF protection, token storage, XSS prevention
