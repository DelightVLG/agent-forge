# Monorepo

Skill для работы с монорепозиториями.

## Темы

- pnpm workspaces: конфигурация, фильтрация команд
- Shared packages: типы, утилиты, конфигурации
- Зависимости между apps: internal packages, workspace protocol
- Версионирование: changesets, independent vs fixed versions
- Build order и topological sort
- Конфигурация TypeScript: project references, composite
- Общие конфиги: ESLint, Prettier, tsconfig через extends
- CI оптимизация: affected-only builds, кэширование
