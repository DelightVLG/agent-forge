# Accessibility Baseline

Every interactive element must:
- Have an accessible name (label, aria-label, or visible text)
- Be keyboard reachable (Tab) and operable (Enter/Space)
- Show a visible focus indicator
- Meet WCAG AA contrast (4.5:1 for text)

Common failures to catch in review:
- `<div onClick>` instead of `<button>`
- Icons without `aria-label`
- Inputs without `<label for>`
- Images without `alt` (use `alt=""` for decorative)
- Color as the only state signal
