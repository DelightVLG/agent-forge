# create-agentforge

Initializer for [`agentforge`](https://www.npmjs.com/package/agentforge). Lets you run:

```bash
pnpm create agentforge my-app
npm create agentforge my-app
yarn create agentforge my-app
```

Under the hood it forwards everything to `agentforge new <args>`. All CLI flags of `agentforge new` (`--template`, `--git`, `--install`, `--yes`, `--lang`) are supported.

See the main package for full docs: https://github.com/delightvlg/agentforge
