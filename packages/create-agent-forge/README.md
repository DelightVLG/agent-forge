# @delightvlg/create-agent-forge

Initializer for [`@delightvlg/agent-forge`](https://www.npmjs.com/package/@delightvlg/agent-forge). Lets you run:

```bash
pnpm create @delightvlg/agent-forge my-app
npm create @delightvlg/agent-forge my-app
yarn create @delightvlg/agent-forge my-app
```

Under the hood it forwards everything to `agentforge new <args>`. All CLI flags of `agentforge new` (`--template`, `--git`, `--install`, `--yes`, `--lang`) are supported.

See the main package for full docs: https://github.com/delightvlg/agent-forge
